Endless Trucker

Unity project focused on optimization, systems design, and stable high-speed gameplay

Unity C# Optimization Systems Design

Project Overview

Endless Trucker is an active game development project started in February 2026. The focus is building performant gameplay systems that support smooth world generation and responsive vehicle movement at high speeds.

The project emphasizes practical systems engineering in Unity: reducing redundant computation, improving runtime consistency, and keeping frame pacing stable as game speed and spawn load increase.

Key Contributions

Lane Spawn Caching

Designed and optimized lane-spawn logic using a caching approach to remove redundant calculations and improve runtime performance.

Spline-Based Vehicle Motion

Developed spline-driven movement for reliable road-aligned behavior, reducing manual setup and improving pathing stability.

Performance Bottleneck Reduction

Identified and improved costly object spawning and update loop paths, resulting in steadier frame rates during high-speed gameplay.
