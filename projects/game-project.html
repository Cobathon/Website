Overview

Project Promotion is a strategic turn-based RPG featuring intelligent AI systems and complex game mechanics. As an AI and Gameplay Programmer, I implemented core systems including behavior tree architectures, save/load progression, level progression mechanics, and cutscene controllers. The game showcases A* pathfinding for enemy navigation, a Conditional Turn-Based (CTB) system using min-heap data structures, and modular behavior trees for flexible character decision-making.

Challenges & Solutions

Learning AI and Behavior Trees

Challenge: AI programming and behavior trees were completely new concepts requiring understanding of design patterns and best practices.

Solution: Extensive research, trial-and-error prototyping, and iterative development starting with simple behaviors and gradually increasing complexity.

Impact: Successfully implemented robust behavior tree system supporting complex AI behaviors with modular, extensible architecture.

Save/Load System Design

Challenge: Preserving complex game state including progression, character data, and AI states with data integrity.

Solution: Centralized SaveManager with Unity serialization, version management, and defensive programming for edge cases.

Impact: Reliable save/load functionality with extensible design supporting future system additions.

System Integration

Challenge: Integrating multiple systems without tight coupling that would hinder maintainability.

Solution: Event-driven architecture with C# delegates/events, component-based design, and clear separation of concerns.

Impact: Systems can be developed and tested independently, enabling parallel team development.

Technical Systems

Behavior Tree Architecture

Hierarchical node structure with composite nodes (selectors, sequences) controlling flow and leaf nodes (actions, conditions) performing tasks. Enables complex AI through composition of reusable components.

Modular Nodes State Management Extensible Design

Save/Load System

Centralized SaveManager coordinating serialization across all game systems. Uses Unity's serialization with custom data structures, version management, and error handling.

Centralized Manager Version Control Error Handling

Cutscene Controller

Event-driven architecture managing narrative sequences and gameplay transitions. Integrates seamlessly through well-defined interfaces and supports branching narratives.

Event-Driven Timeline-Based Branching Support

A* Pathfinding

Priority queue (min-heap) implementation for optimal path calculation. Considers terrain costs and movement constraints for intelligent AI navigation.

Optimal Paths Terrain-Aware Real-Time Performance
