Turn-based RPG with A* pathfinding AI, conditional turn-based system, and modular behavior trees

Unity C# A* Pathfinding AI Programming Game Development

Project Overview

Developing a sophisticated turn-based RPG in Unity that showcases advanced AI programming and game systems design. The project demonstrates my ability to implement complex game mechanics including intelligent enemy AI, dynamic turn management, and modular character behavior systems.

The game features enemies with A* pathfinding-based AI that intelligently navigate the battlefield, a Conditional Turn-Based (CTB) system using min-heap data structures for efficient turn order management, and behavior trees that enable flexible and maintainable character decision-making logic.

Started in January 2025 and currently in active development, this project combines my computer science background with game design skills from my minor, resulting in a technically sophisticated RPG with engaging gameplay mechanics.

Key Features

A* Pathfinding AI

Intelligent enemy AI using A* algorithm for optimal path planning, allowing enemies to navigate complex battlefields and position strategically.

Conditional Turn-Based System

Engineered a CTB system with min-heap data structure for O(log n) turn order updates, ensuring efficient performance even with many active characters.

Modular Behavior Trees

Implemented behavior tree architecture with state machines for character decision-making, enabling complex AI behaviors through composable nodes.

Dynamic Turn Order UI

Created UI-linked timeline that visualizes turn order in real-time, updating dynamically as character speeds and actions affect initiative.

Performance Optimization

Balanced pathfinding accuracy with performance to enable real-time AI planning in large maps without frame drops or lag.

State Machine Integration

Integrated finite state machines with behavior trees to manage character states (idle, moving, attacking) with clean transitions.

Technical Implementation

Behavior trees structure AI decision-making through composite nodes (selectors, sequences) and leaf nodes (actions, conditions). Each enemy type has a customizable behavior tree defining their combat strategy aggressive enemies prioritize attacking, defensive enemies seek cover, and support enemies heal allies. The modular design allows rapid prototyping of new AI personalities without code changes.

Game Design & Systems

Strategic Positioning

AI considers tactical positioning including range management, flanking opportunities, and terrain advantages when planning movements and attacks.

Combat Mechanics

Turn-based combat with speed-based initiative, action point systems, and status effects that influence turn order and tactical options.

Scalable AI

Behavior tree architecture allows easy creation of new enemy types by mixing and matching behavior nodes, enabling rich enemy variety.

Visual Feedback

Clear UI indicators for movement ranges, attack zones, and turn order help players make informed tactical decisions during combat encounters.

Development Progress & Future Plans

The core systems pathfinding AI, CTB turn management, and behavior trees are fully implemented and functional. Current development focuses on expanding content including new character abilities, enemy types, and combat encounters that showcase the AI system's capabilities.

Future enhancements include cooperative multi-agent AI where enemies coordinate attacks, improved pathfinding with formation movement, and advanced behavior tree debugging tools. I'm also exploring Monte Carlo Tree Search (MCTS) for boss AI that can predict player strategies and adapt accordingly.

This project demonstrates not just technical proficiency in C# and Unity, but also my ability to design complex interconnected game systems that are both performant and maintainable. The combination of computer science algorithms with game design principles results in engaging, intelligent gameplay.
